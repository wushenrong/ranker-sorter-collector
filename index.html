Ranker Sorter with Data Collection

An example project that will send results to Google Sheets using Apps Scripts. Backend code can be found inside the backend directory the project's GitHub repository page, or inside the Apps Script project page.

This site does not save any data on your computer. Do not reload the browser while completing the ranker. Do not forget to save your results after completing the ranker!

Made with open source projects. Based on Ranker Sorter.
